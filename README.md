# 📥 คู่มือดาวน์โหลดวิดีโอ/เพลงจาก YouTube บน iPhone และ iPad

ใช้ร่วมกันระหว่างแอป **a-Shell**, **Shortcuts** และ **VLC** เพื่อดาวน์โหลดและเล่นไฟล์วิดีโอ/เพลงจาก YouTube โดยไม่ต้องใช้คอมพิวเตอร์

> ⚠️ **ข้อควรระวัง**: โปรดใช้เพื่อดาวน์โหลดเฉพาะเนื้อหาที่ท่านมีสิทธิ์ใช้งาน เช่น วิดีโอของตนเอง เนื้อหาที่ได้รับอนุญาต หรือเนื้อหาที่เป็นสาธารณสมบัติ การดาวน์โหลดวิดีโอที่มีลิขสิทธิ์โดยไม่ได้รับอนุญาตอาจขัดต่อข้อกำหนดการใช้งานของ YouTube และกฎหมายลิขสิทธิ์ในบางประเทศ

---

## 🧩 สิ่งที่ต้องเตรียม

| แอป | หน้าที่ |
|---|---|
| **a-Shell** | Terminal บน iOS สำหรับรันคำสั่ง `yt-dlp` |
| **Shortcuts** | รับลิงก์จาก YouTube แล้วส่งต่อไปยัง a-Shell |
| **VLC** (ทางเลือก) | เล่นไฟล์วิดีโอ/เพลงที่ดาวน์โหลดมาแล้ว |

---

## ติดตั้ง a-Shell และ yt-dlp

1. ดาวน์โหลดแอป **a-Shell** จาก App Store
2. เปิดแอป แล้วติดตั้ง `yt-dlp` ด้วยคำสั่ง:

   ```bash
   pip install yt-dlp
   ```

3. ตรวจสอบว่าติดตั้งสำเร็จ:

   ```bash
   yt-dlp --version
   ```
